## 📌 스터디 규칙  
- 스터디 시간: **매주 화요일 21시**, Google Meet 화상회의  
- 모든 스터디원은 스터디 시작 전까지 자료 준비 (형식: Notion, Word, ipynb파일 등 자유)  
- 세미나 방식으로 번갈아가며 발표 → 질의응답 진행  
- 다음주차 스터디 주제는 스터디 시간중에 선정
<br>


## 📝 보고서  
- 팀원들이 제출한 보고서를 취합하여 양식에 맞춰 작성 후 제출, GitHub 레포지토리 업로드  
- 보고서 작성 순서: **정찬원 → 임소정 → 서준원 → 송주훈 → 강인우 → 정민섭**
